# Ai-Business-Insights-and-Decision-Engine-NLP-Project

A project focused on NLP for business insights and decision making.

## Project Structure
- `src/`: Source code for data processing and modeling.
- `data/`: Dataset directory.

## Setup
1. Clone the repository.
2. Install dependencies (spaCy, pandas, etc.).
3. Download the spaCy model: `python -m spacy download en_core_web_sm` (or the script will do it automatically).

## Usage
Run `python src/preprocessing.py` to process the data.
